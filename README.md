## 個人自介網站

這是一個介紹我本人的經歷的網站，此專案是使用 PHP+Laravel 來進行開發，並使用 Dcat Admin 這個 laravel 後台套件來進行網站後台的開發，便於我管理網站前台該呈現的資料。前台的部分則是使用 HTML5 的模板來進行網站畫面的呈現。

## 開發環境

PHP => 7.2.5<br>
Laravel => 7.30.6
